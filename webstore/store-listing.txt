============================================================
Chrome Web Store 掲載情報（コピペ用）
============================================================

【ストアの掲載情報】タブ
------------------------------------------------------------

■ 名前（30文字以内）
いろいろMDコピー

■ 概要（132文字以内）
GitHub / Azure DevOps / AWS CodeCommit の PR を Markdown、SharePoint の会議録画から VTT 字幕、Teams チャットを Markdown でワンクリック取得できる Chrome 拡張。

■ 説明（16,000文字以内）

複数サイトの情報を Markdown / VTT でダウンロードできる Chrome 拡張機能です。
GitHub・Azure DevOps の PR レビュー、SharePoint Stream の Teams 会議録画字幕（VTT）、
そして Microsoft Teams のチャット履歴まで、ワンクリックで手元に保存できます。

【できること】

操作はすべて、ツールバーのアイコンをクリックして開くポップアップから行います。
ポップアップは今いるサイトを判定し、そのサイトに合ったボタンを自動で出し分けます。

▼ Markdown でダウンロード / コピー（PR レビュー）
GitHub / Azure DevOps の PR ページでアイコンをクリックすると、ポップアップに
「MDでダウンロード」「MDコピー」が出ます。タイトル・本文・全レビューコメントを
1つの .md ファイルで保存、またはクリップボードにコピーできます。

▼ PR 一覧ページからも直接ダウンロード
PR 一覧ページでは各行にダウンロードボタンが表示され、PR を開かずに直接取得できます。

▼ レビューコメントを完全に取得
投稿者名・日時・対象ファイルパス・差分コード（diff）付きで取得します。
インラインコメント、ファイル単位コメント、全体コメントのすべてに対応しています。
Azure DevOps では DOM から取得できない場合に REST API へ自動フォールバックします。

▼ GitHub / Azure DevOps / AWS CodeCommit 対応
・GitHub（github.com）
・Azure DevOps（dev.azure.com / *.visualstudio.com）
・カスタムドメインの Azure DevOps
・AWS CodeCommit（AWS マネジメントコンソールの PR 詳細ページ。詳細ページ専用）
SPA（シングルページアプリケーション）のページ遷移にも追従します。

▼ SharePoint Stream（Teams 会議録画）対応
会議録画ページでアイコンをクリックすると、ポップアップに「VTTでダウンロード」「VTTコピー」が
出ます。トランスクリプト（字幕）を .vtt で保存／コピーできます。議事録作成の下書きに便利です。

▼ Microsoft Teams チャット対応
Teams（teams.microsoft.com / teams.live.com / teams.cloud.microsoft）のチャット・チャネルで
アイコンをクリックすると、ポップアップに次のボタンが出ます。
・「MDでダウンロード」「MDコピー」: 自動スクロールで全履歴を収集し、送信者・日時・本文・
  リアクション・添付リンクを 1つの .md ファイルで保存／コピー。

【こんな場面で役立ちます】
・コードレビューの記録をチームで共有したい
・PR の内容をドキュメントや議事録に転記したい
・レビューの振り返り資料を作りたい
・複数の PR をまとめて Markdown ファイルで保存したい
・Teams 会議の内容を字幕データから議事録に起こしたい
・Teams のチャット履歴をバックアップ・記録として残したい

【使い方】
1. Chrome Web Store からインストール
2. 対応ページ（GitHub / Azure DevOps の PR、SharePoint Stream の会議録画、Microsoft Teams の
   チャット・チャネル）を開く
3. ツールバーの拡張機能アイコンをクリック → ポップアップにそのサイト用のボタンが表示されます
   - PR: 「MDでダウンロード」「MDコピー」
   - SharePoint 会議録画: 「VTTでダウンロード」「VTTコピー」
   - Teams チャット: 「MDでダウンロード」「MDコピー」
   ※ PR 一覧ページでは各行のダウンロードボタンからも直接保存できます
4. ボタンをクリック → .md / .vtt ファイルが保存（またはコピー）されます

【プライバシーについて】
・個人情報の収集は一切行いません
・外部サーバーへのデータ送信はありません
・PR ページ / SharePoint Stream / Teams ページの DOM・公式 API の読み取りと、
  クリップボードへの書き込み・ファイルのダウンロードのみを行います
・Teams の添付取得はユーザー自身の既存ログインセッションを用い、Microsoft 関連ドメインに
  限定して行います（第三者への送信はありません）
・すべてのコードは拡張機能パッケージ内に含まれています

【プライバシーへの取り組み】タブ
------------------------------------------------------------

■ このアイテムに単一の目的はありますか？
はい

■ 単一の目的の説明
閲覧中のページ（PR レビュー / Teams 会議録画 / Teams チャット）の内容を Markdown・VTT 形式でダウンロードまたはコピーする

■ データの使用に関する開示

  ・個人を特定できる情報を収集しますか？
    いいえ

  ・ユーザーのデータをどのように扱いますか？
    - ユーザーデータの販売: しない
    - ユーザーデータの本来の目的以外への使用や転送: しない
    - 信用力の判定や貸付目的での使用や転送: しない

■ 権限の正当性

  activeTab:
  ユーザーがポップアップから操作した際に、現在のタブの対応ページ
  （PR / SharePoint 会議録画 / Teams チャット）の内容を読み取り、
  Markdown・VTT 形式へ変換するために使用します。

  scripting:
  カスタムドメインのAzure DevOpsや、別サービスから遷移したAWS CodeCommitの
  ページにコンテンツスクリプトを動的に注入するために使用します。

  webNavigation:
  SPA（シングルページアプリケーション）のページ遷移を検出し、
  PR一覧の行ダウンロードボタンの再表示やコンテンツスクリプトの
  再初期化を適切なタイミングで行うために使用します。

  clipboardWrite:
  ポップアップの「コピー」操作で、整形した Markdown / VTT テキストを
  クリップボードに書き込むために使用します（読み取りは行いません）。

  host_permissions (github.com / *.github.com / dev.azure.com / *.visualstudio.com):
  PRページのタイトル・本文・レビューコメントを読み取り、Markdown に整形するために使用します。

  host_permissions (console.aws.amazon.com / *.console.aws.amazon.com):
  AWS マネジメントコンソールの CodeCommit PR 詳細ページからタイトル・本文・
  レビューコメントを読み取り、Markdown に整形するために使用します。

  host_permissions (*.sharepoint.com):
  SharePoint Streamの会議録画ページで、トランスクリプト (VTT 字幕) 取得用の
  Microsoft 公式 API を呼び出すために使用します。

  host_permissions (teams.microsoft.com / *.teams.microsoft.com / teams.live.com / teams.cloud.microsoft):
  Microsoft Teams のチャット/チャネルでメッセージ履歴を読み取り、Markdown に整形するために使用します。

  optional_host_permissions (カスタムドメイン):
  社内ホストされた Azure DevOps などのカスタムドメインを個別に許可していただくために、
  拡張機能アイコンのポップアップから「このサイトで動作を許可する」ボタンを押した場合のみ、
  そのオリジンに対してのみ権限を取得します。ユーザーが明示的に許可していない
  ドメインでは一切動作しません。

■ リモートコードを使用していますか？
いいえ
（すべてのコードは拡張機能パッケージ内に含まれており、外部からのコード読み込みは一切行いません。）
