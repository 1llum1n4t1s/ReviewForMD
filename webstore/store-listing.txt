============================================================
Chrome Web Store 掲載情報（コピペ用）
============================================================

【ストアの掲載情報】タブ
------------------------------------------------------------

■ 名前（30文字以内）
プルリクMDコピー

■ 概要（132文字以内）
GitHub / Azure DevOps の PR 内容を Markdown ファイルでダウンロード。タイトル・本文・レビューコメントをワンクリックで取得できます。

■ 説明（16,000文字以内）

プルリクエスト（PR）の内容を Markdown 形式でダウンロード＆コピーできる Chrome 拡張機能です。
GitHub と Azure DevOps の両方に対応しています。

【できること】

▼ Markdown ファイルでダウンロード
PR 詳細ページの「MDでダウンロード」ボタンを押すと、タイトル・本文・全レビューコメントを
1つの .md ファイルとしてダウンロードできます。ファイル名には PR タイトルが自動で使われます。

▼ PR 一覧ページからも直接ダウンロード
PR 一覧ページの各行にもダウンロードボタンが表示されます。
わざわざ PR を開かなくても、一覧から直接 Markdown ファイルを取得できます。

▼ 個別コメントのコピー
各レビューコメントに「MDコピー」ボタンが追加されます。
特定のコメントだけを Markdown テキストとしてクリップボードにコピーできます。

▼ レビューコメントを完全に取得
投稿者名・日時・対象ファイルパス・差分コード（diff）付きで取得します。
インラインコメント、ファイル単位コメント、全体コメントのすべてに対応しています。
Azure DevOps では DOM から取得できない場合に REST API へ自動フォールバックします。

▼ GitHub & Azure DevOps 両対応
・GitHub（github.com）
・Azure DevOps（dev.azure.com / *.visualstudio.com）
・カスタムドメインの Azure DevOps
SPA（シングルページアプリケーション）のページ遷移にも追従します。

【こんな場面で役立ちます】
・コードレビューの記録をチームで共有したい
・PR の内容をドキュメントや議事録に転記したい
・レビューの振り返り資料を作りたい
・複数の PR をまとめて Markdown ファイルで保存したい

【使い方】
1. Chrome Web Store からインストール
2. GitHub または Azure DevOps の PR ページを開く
3. 「MDでダウンロード」ボタンをクリック → .md ファイルが保存されます

【プライバシーについて】
・個人情報の収集は一切行いません
・外部サーバーへのデータ送信はありません
・PR ページの DOM 読み取りとクリップボードへの書き込みのみを行います
・すべてのコードは拡張機能パッケージ内に含まれています

【プライバシーへの取り組み】タブ
------------------------------------------------------------

■ このアイテムに単一の目的はありますか？
はい

■ 単一の目的の説明
プルリクエストのタイトル・本文・レビューコメントをMarkdown形式でダウンロードまたはコピーする

■ データの使用に関する開示

  ・個人を特定できる情報を収集しますか？
    いいえ

  ・ユーザーのデータをどのように扱いますか？
    - ユーザーデータの販売: しない
    - ユーザーデータの本来の目的以外への使用や転送: しない
    - 信用力の判定や貸付目的での使用や転送: しない

■ 権限の正当性

  activeTab:
  ユーザーがボタンをクリックした際に、PRページのDOM要素を読み取り
  Markdown形式に変換するために使用します。

  scripting:
  カスタムドメインのAzure DevOpsページにコンテンツスクリプトを
  動的に注入するために使用します。

  webNavigation:
  SPA（シングルページアプリケーション）のページ遷移を検出し、
  適切なタイミングでボタンを注入するために使用します。

■ リモートコードを使用していますか？
いいえ
（すべてのコードは拡張機能パッケージ内に含まれており、外部からのコード読み込みは一切行いません。）
