# プライバシーポリシー - プルリクMDコピー (ReviewForMD)

最終更新日: 2026-04-18

## 概要

「プルリクMDコピー」は、ユーザーのプライバシーを最大限に尊重するChrome拡張機能です。

## データの収集

この拡張機能は、いかなる個人情報も収集、保存、送信しません。

## 使用する権限

### activeTab
ユーザーがツールバーアイコンをクリックした際、または PR ページのボタンをクリックした際に、現在のタブからプルリクエストの内容を取得するために使用します。

### scripting
カスタムドメインの Azure DevOps ページに対して動的にコンテンツスクリプトを注入するために使用します。

### webNavigation
SPA ナビゲーション（GitHub の Turbo、Azure DevOps の React ルーティング）を検出して、適切にボタンを再注入するために使用します。

## ホスト権限

### github.com / *.github.com / dev.azure.com / *.visualstudio.com / *.sharepoint.com
プルリクエストページ（GitHub / Azure DevOps）から内容を抽出する、または SharePoint Stream ページからトランスクリプト（VTT）を取得するために使用します。

## 外部通信

この拡張機能は、外部サーバーへの通信を一切行いません。GitHub / Azure DevOps / SharePoint への通信は、ユーザーが開いているサービスに対するもののみで、いずれもユーザー自身が認証済みのオリジン内に閉じています。

## リモートコード

この拡張機能は、外部からのコード読み込みを一切行いません。すべてのコードは拡張機能パッケージ内に含まれています。
