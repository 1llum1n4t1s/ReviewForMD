■ 名前
いろいろMDコピー

■ 概要
GitHub / Azure DevOps / AWS CodeCommit の PR を Markdown、SharePoint の会議録画から VTT 字幕、Teams チャットを Markdown でワンクリック取得できる拡張機能。

■ 説明
複数サイトの情報を Markdown / VTT でダウンロードできる拡張機能です。
GitHub・Azure DevOps・AWS CodeCommit の PR レビュー、SharePoint Stream の Teams 会議録画字幕（VTT）、
そして Microsoft Teams のチャット履歴まで、ワンクリックで手元に保存できます。

【できること】

操作はすべて、ツールバーのアイコンをクリックして開くポップアップから行います。
ポップアップは今いるサイトを判定し、そのサイトに合ったボタンを自動で出し分けます。

▼ Markdown でダウンロード / コピー（PR レビュー）
GitHub / Azure DevOps / AWS CodeCommit の PR ページでアイコンをクリックすると、
タイトル・本文・全レビューコメントを 1 つの .md ファイルで保存、またはクリップボードにコピーできます。
PR 一覧ページ（GitHub / Azure DevOps）では各行のボタンから、PR を開かずに直接ダウンロードもできます。

▼ レビューコメントを完全に取得
投稿者名・日時・対象ファイルパス・差分コード（diff）付きで取得します。
インラインコメント、ファイル単位コメント、全体コメントのすべてに対応しています。
Azure DevOps では DOM から取得できない場合に REST API へ自動フォールバックします。

▼ SharePoint Stream（Teams 会議録画）対応
会議録画ページでアイコンをクリックすると、トランスクリプト（字幕）を .vtt で保存／コピーできます。
議事録作成の下書きに便利です。

▼ Microsoft Teams チャット対応
Teams のチャット・チャネルを自動スクロールで全履歴収集し、送信者・日時・本文・リアクション・添付リンクを
1 つの .md ファイルで保存／コピーできます。

【こんな場面で役立ちます】

・コードレビューの記録をチームで共有したい
・PR の内容をドキュメントや議事録に転記したい
・レビューの振り返り資料を作りたい
・Teams 会議の内容を字幕データから議事録に起こしたい
・Teams のチャット履歴をバックアップ・記録として残したい

【プライバシーについて】

・個人情報の収集は一切行いません
・外部サーバーへのデータ送信はありません
・閲覧中ページの DOM・公式 API の読み取りと、クリップボードへの書き込み・ファイルのダウンロードのみを行います
・すべてのコードは拡張機能パッケージ内に含まれています
